Appointment Scheduler 2.0

This application is designed to allow for a company to set and track customer appointments.
The user can create, update, and delete both customers and appointments and all data is saved to a database.
In order to run the program the user must enter a valid username and password and login.  Once logged in the user can
choose from a variety of options to view and modify customers and appointments as well as view custom reports.
The additional Report included displays customers by country and region allowing the company to see where the densest
populations are.
